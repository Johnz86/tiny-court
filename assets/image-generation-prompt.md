### The "Tiny Court" Prompt System

This system uses a fixed **Static Scene** definition that remains identical, combined with a variable **Mobile Drama** definition that you replace for each new scenario. All character and object descriptions *must* include coordinates to ensure they occupy the correct spatial "anchors."
